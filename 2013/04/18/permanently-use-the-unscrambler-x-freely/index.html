Permanently Use The Unscrambler X 10.2 Trial Version Freely

Posted on 2013-04-18 | In 精品推介

The Unscrambler X 10.2软件在化学计量学中获得了广泛应用，尤其是其多元统计分析的能力，更是极大地方便了科研工作者。然而，比较遗憾的是，一套The Unscrambler X 10.2要上万美元，即使是面向院校师生的学生版也要将近6000美元；尤其是一套软件只能安装到一台电脑上，更是限制了用户购买的热情，以至于其在中国国内并没有获得应有的广泛运用。 所以，每每看到科研人员不能享受这一工具带来的便捷时，月夜心生遗憾。不过这也难怪，当前网络上只有The Unscrambler 的9.7版本的破解版可供下载，而The Unscrambler X 10.2的试用版只能使用30天，过期之后，便不得不忍痛舍弃。可是，9.7版跟X 10.2版的差距是显然的，功能没有后者强大不说，即使是视觉效果也差了一个档次。 那么，当我们的电脑试用The Unscrambler X 10.2超过30天之后，除了购买，真的没有办法继续使用了吗？或许并不尽然。 The Unscrambler X 10.2的登录验证界面如下：

我们使用试用版的时候，需要首先点击下面的Obtain连接到The Unscrambler X 10.2的网站上申请一个激活码，而申请的激活码似乎只跟机器码（Machine ID）有关。我们自然知道，这个机器码是跟本地计算机有关的，使用这个机器码申请的激活码过期之后，我们可以修改这个机器码，然后重新申请一个激活码。 不过遗憾的是，目前想找到一款可以修改机器码的好软件却并不容易。因此，这一方法只好作罢。 除此之外，月夜认为可能系统中应该有记录The Unscrambler X 10.2软件使用日期的东西，当然，这些东西是在注册表里。下面我们就开始处理。 首先卸载The Unscrambler X 10.2软件，然后使用regedit命令打开注册表，以关键词“unscrambler”搜索 _所有 _注册表项，并将其删除。完成这些步骤之后，重启计算机，接着重装The Unscrambler X 10.2。完成之后打开The Unscrambler X 10.2，同样可以看到上面的验证注册窗口，在其中输入我们以前申请到的激活码，然后点击Activate，可以看到，对话框消失，我们熟悉的The Unscrambler X 10.2软件界面出现在了我们的面前。 如此，我们便可以继续免费地试用The Unscrambler X 10.2试用版，继续使用它助力我们的科研。
