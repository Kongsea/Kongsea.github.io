使用Dandelion主题建立WordPress儿童博客

Posted on 2013-06-07 | In 网站相关

此前 月夜 曾经介绍过使用WordPress的Bodega主题建立儿童博客的教程，虽然那款主题在一定程度上也算美观，而且能做出不错的效果，还算适宜用来做儿童博客。但在使用那款主题的过程中，月夜总是觉得其外观看起来不够完美，而后台的设置和添加内容也不够方便，因此一直以来都在考虑更换主题，重新构建。 前几天在网络上无意间碰到了一款名为Dandelion的主题，这款主题的名称的汉语意思是蒲公英，如名所言，其给人的第一印象就是一种小清新的气息扑面而来。而最终构建的博客效果朋友们可以先预览一下：

朋友们可以大体预览一下，使用这款主题建立的儿童博客效果十分不错，多幅照片的展示也能最大程度上体现出儿童博客的关注点。 下面，我们就简单地说一下整个博客的建立中有关主题的内容。

1.博客头部幻灯片

整个博客中最引人注意的毫无疑问是首页中大幅图片展示的一系列幻灯片效果。而这些图片的定制性十分容易，该主题在后台提供了这些设置。具体的过程如下：

1.1 设置幻灯片

进入WordPress后台，在“外观”菜单下找到“Dandelion Options”子菜单，我们直入主题地进入主题设置中的Slider Settings，也即幻灯片设置菜单。 这里提供了一共四种幻灯片的形式，而我们上面的效果中所使用的则是其中的Thumbnail Slider，即缩略滑块形式的幻灯片。 这里的设置中，我们选中了Automatic image resizing和Autoplay，也即自动缩放图片和自动播放；而下面的两个间隔Interval取默认设置即可。Pause on hover即鼠标悬停暂停播放也选中了。这是这些基本设置。

1.2 创建幻灯片

设置完之后，我们可以在设置页面的下方Add new thumbnail slider里新建一个缩略滑块幻灯片，这里我们取名为TS，创建以后，我们点击页面最下方的“Save Changes”将其保存。需要注意的是，所有在此页面进行的操作，在操作完毕之后，都要记得点击Save Changes进行保存，不然改动会丢失。

1.3 为幻灯片添加图片

保存之后，页面上（可能需要刷新）会有Add image to TS栏目，如下图所示：

点击如上图所示的Upload按钮，即可上传图片，当然，你也可以在第二栏里为上传的图片设置一个链接，以便在首页的幻灯片里点击这张图片时跳转到一个新的地方（比如设置为一篇文章的地址，跳转到该文章介绍这张图片的来源背景等等）。此外，在第三栏里可以添加描述，内容会在首页播放幻灯片时显示在图片上。设置完毕，点击Add image按钮，即可完成添加。 需要注意的是，添加成功后，页面上没有提示，所以你可以下拉看看幻灯片的图片库，通常都是直接添加成功的。而且，添加成功后，页面没有变化，所以如果要接着添加图片，你需要手动将上面栏目里的内容清空，然后再次点击Upload上传图片，设置描述，Add image。

1.4 修改幻灯片及图片

最后，当你把需要添加的所有图片都添加完以后，还可以在下面的幻灯片图库里用鼠标拖动而改变幻灯片图片之间的顺序：

如上图所示，除了拖动改变图片顺序，还可以删除或修改已经添加的图片；修改或删除已经创建的幻灯片；或者新建一个Thumbnail类型的幻灯片。 最后需要特别说明的是，在进行所以以上操作之后，你都不要忘记最后点击Save Changes来保存操作，否则，这些操作都将无用。

2.博客中部图片

除了博客头部的幻灯片之外，该主题的另一个特色是其在首页中部所显示的三幅图片，这三幅图片在Dandelion主题中也可以随时设置修改与替换。不过这个过程比较简单，我们一笔带过。 跟1中一样，首先进入WordPress后台的“外观”菜单下找到Dandelion Options子菜单，进入其中的Page Settings，然后设置其中的Home选项标签，这里分别提供了所有三幅图片的设置选项栏。我们以其中的第三幅图为例来进行查看，如下图：

很明显，这里提供了第三幅图（Third Box）的标题、描述、图片、按钮文本已经链接五项内容。上传图片，设置选项，加上链接，即可完成设置。 不过需要注意的是，跟上面一样，设置完毕，最后还需要点击Save Changes来保存设置。

3.上传相册

对于一个儿童博客来说，上传大量的图片会是必然，而显然，仅在首页显示的一个幻灯片和其下的几张图片根本不能满足需求。那么，我们如何上传更多的图片呢？Dandelion主题提供了功能来满足这一需求。 首先还是进入WordPress后台，点击Portfolio（作品）菜单下的Add New子菜单添加一个新的Portfolio。 为新上传的图片添加题目和内容，然后点击下面的DANDELION PORTFOLIO ITEM SETTINGS中的Preview Image URL右侧的Upload按钮，上传图片。完成之后，点击右侧的“发布”按钮，即可将图片发布出去。 当然，你也可以对其他的设置选项根据自己的需要进行灵活设置。而且，推荐诸位为所有的Portfolio设置一些分类，方便归类组织。

4.新建页面

上传完毕，我们需要设置一下博客的展示。比如哪个页面展示博客内容，哪个页面展示相册内容，等等。 具体的过程如下： 进入WordPress后台，点击“页面”菜单下的“新建页面”子菜单新建一个页面。页面取名“我的博客”（类似均可），不用添加任何内容，只需要在右侧的“页面属性”中的“模板”里选择“Featured page”，然后发布页面即可。这样的一个页面就是专门用来收拢展示所有博客内容的容器。 而新建另一个页面，取名“我的相册”（类似），不添加任何内容，而在右侧的页面属性中的模板里选择“Portfolio Gallery”，发布页面，就会得到一个类似如下图所示的页面：

当然，你也可以将页面属性设置为“Portfolio Page”，这样可以得到如下图所示的效果：

你可以根据你的喜好自由选择。不过显然，Portfolio Gallery的容量更大。 这是另一个问题，我不知道通常情况下WordPress默认是否已经创建了“首页”这个页面，如果没有，你也需要手动添加一个页面，取名为首页，不要添加任何内容，然后选择模板为“Home Page”进行发布。

5.显示设置

虽然我们已经进行了上面的所有设置，但最后还需要一步。从WordPress后台进入“设置”菜单下的“阅读”子菜单，从中找到“首页显示”，选择首页显示方式为 一个静态页面，将“主页”设置为“首页”，而将文章页设置为“我的博客”。最后“保存更改”，即可完成所需要的设置。 而经过以上数步的操作，我们已经基本上完成了WordPress主题Dandelion的各项基本设置，建立了一个儿童博客。而为其中添加文章，添加图片，便可以日渐扩充内容，渐渐地记录着孩子的生活与成长，一天天完善着你的博客。
