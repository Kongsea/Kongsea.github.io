推荐几款简洁朴实的免费英文字体

Posted on 2013-04-25 | In 精品推介

前一段时间在为MATLAB寻找合适的字体时，微软正黑体的英文字体并不出众，所以一直在搜寻一款英文字体，以便能跟微软正黑体一起组合在MATLAB中使用。我们是程序设计，因此使用Google搜索了“program font free”，然后据此又找了一些其他的英文字体。这些字体，大多是从国外类似“50 free fonts for ”的文章中一款款挑选出来，不过，因为我们的具体用途，也因为我的性格，有些夸张的字体肯定只能刨去，而有些0和O、1和l不能很好地加以区分的字体也尽量不用。 经过我的逐个挑选之后，最终选出了几款我认为十分不错的英文字体，这些字体刻板而不失灵巧，庄重而有着活泼，款款皆为我所认为的精品。可是收藏在我的电脑上终究没有什么作用，因此今天整理一下发出来，以供后来者参考。

1.开源的等宽字体Inconsolata

这款字体是月夜目前的默认英文字体，配合微软正黑体使用，中英得当，效果很好。

2.Adobe发布的等宽字体Source Code Pro

或许从名字你也能看出来，这是Adobe公司专为程序员们设计的一款字体。

3.带着活泼修饰的Museo

跟前两种字体不同，这种字体不是等宽字体，但它的适度修饰却把整款字体都给带的活泼可爱起来了。

4.极细字体Nixie One、Quicksand和NotCourierSans

这几种字体都有着极细的笔画，不过它们之间又有很多区别。喜欢这种极细风格的朋友可以对比选择一下。

5.中规中矩的Lucida、Ledger、Jura、Contra和Apparatus

这几种字体比较相像，其中Lucida Fax为系统内预装的字体，朋友们可以直接查看效果。其均比较中规中矩，略有衬线，对数字和字母均没有进行特定的处理。喜欢这类没有任何出格的字体的朋友可以对比下载。
