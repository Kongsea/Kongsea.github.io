Windows 7 Ultimate Build 7100 RC英文版安装小记

Posted on 2009-04-25 | In 时讯品评

最近几天微软放出了Windows 7 Ultimate的RC英文版，版本号是Build 7100。作为正式版的前奏，以及微软Windows 7的第一个RC版，月夜 自然对此也十分关注，并在第一时间下载了x86版本，在虚拟机上简单安装了一下。虽然RC版本和以前RC分支系列的7086以及RTM分支系列的7106等版本都没有太大区别，但月夜考虑到很多朋友没有下载安装，可能也会想了解一下这款最新操作系统的最新版本，于是就在安装中简单截取数图，写就此文，以飨我的朋友们。 安装过程依然十分轻松，即使直接硬盘安装，和以前的安装方式也完全相同，更何况此次月夜是通过虚拟机安装，就更是十分简单了。首先是安装中的最开始画面：

当然，这个截图我们几乎都已经看厌了，唯一的变化无非是语言。接下来的界面也没有什么变化：

此RC版本安装过程中和以前相比最大的变化在此：

可以看到，在这里我们可以选择安装Windows 7的各种版本，比如Windows 7 Starter、Windows 7 Home Basic、Windows 7 Professional、Windows 7 Ultimate等等。尽管我不清楚这里面的具体技术，但是否表明微软在Windows 7 正式版中会嵌入所有版本内容，而通过让用户购买相应的激活码来选择使用不同的版本呢？ 下面，我们继续为朋友们送上在Windows 7 RC版安装中的各种截图：

从以上几个截图中我们可以看到Windows 7的启动界面，在最后一个界面上，我们可以看到Ultimate字样。当然，月夜认为，在简体中文版发布时，这里可能会像Build 7106版本那样，使用“旗舰版”代替。 和以前的所有微软操作系统一样，Windows 7 RC版安装过程中，也会多次重启：

在最后一次重启之后，我们正式进入Windows 7 Ultimate Build 7100 RC 英文版的桌面：

当然，这里和以前相比依然没有什么变化。在桌面的右下角，我们可以看到版本号标识：

在这里我们可以看到其版本号：Windows 7 Evaluation copy. Build 7100，其含义为这是一个评估副本。比较有意思的是，从上图我们可以看到，系统时间默认为美国当地时间，而不是计算机内部的时间，或者简体中文版安装时的本地时间。
