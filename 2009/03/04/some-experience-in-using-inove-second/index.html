iNove主题本地化的几点经验(二)

Posted on 2009-03-04 | In 网站相关

在前面的文章 iNove主题本地化的几点经验(一) 中，月夜 为您介绍了自己在将iNove主题本地化时的一些经验，比如拓宽页面、修改字号、修改标题样式、添加postviews插件等等。本文，月夜将接着上文中的话题，继续为您介绍我在iNove主题本地化中的其他一些经验。

与前文相同，月夜使用的iNove主题版本是1.2.3。而且，在我们修改主题文件时，所提及的文件以及文件信息依然指未修改前的绝对位置，而不指修改中的相对位置。

6.修改页面顶端代表首页的“小屋”图标

不知道诸位的感想如何，月夜对那个图标不是太满意，而且还太小，感觉很不醒目。因此月夜想换掉那个图标。

首先找到主题文件夹下img文件夹中的menu.gif图像，使用适当的工具将其上的“小屋”图标换成你想要的图形或文字，并大致估算一下新图标所占据的宽度。然后，我们打开style.css文件，找到其中第225行，将其修改为：

width:80px;

不过，你需要根据自己图标的宽度来适当修改此处的数值；当然，你也可以一点一点的修改，直到你的首页图标位置另你满意为止。

7.修改侧边栏为真正单栏

iNove主题的侧边栏是组织成四部分的，MG12称其为“东西南北”，但月夜觉得似乎更像是打升级的四个人。不管怎么说，月夜觉得中间的两栏并列并不是太好看，所以，就自作主张将其修改成了真正的单栏。

为了使侧边栏真正汉化，月夜将主题文件夹下sidebar.php文件中第65、67行两行中的Recent Posts和Random Posts分别修改为了最新文章和随机文章；将第92行的Recent Comments修改为了最新评论；第102行的Tag Cloud修改为了标签云；第119行的Categories修改为了文章分类；第135行的Archives修改为了文章存档；第153行的Meta修改为了功能。

8.在文章页添加上版权信息

在每一篇文章中都自动添加上一些版权信息，以提示网友在转载时注明出处。我们可以这样在iNove主题中实现。

首先打开style.css文件，在其中第485行之后，添加如下的三行代码：

.reship{ float:left; width:99%; margin-bottom:20px; font-size:12px; }
.reship p{ padding:0px 10px 0px 0px; text-align:right; color:#999999; border-right:1px dashed #BBBBBB; font-size:12px; }
.reship a:link, .reship a:visited, .reship a:hover{ color:#999999; text-decoration:underline;}

然后打开主题文件夹下的single.php文件，找到其中第27行，在其后面添加如下代码：

转载原创文章请注明：文章转载自：<?php bloginfo(‘name’); ?> [<?php echo get_settings(‘home’); ?>]

本文标题：<?php the_title(); ?>

本文地址：<?php the_permalink(); ?>



这样，我们就成功在博客每一篇文章的后面添加上了版权信息。当然，你也可以修改上面的代码来添加上属于自己的个性版权信息。

9.修改底部的一些信息

因为iNove主题的架构和一般主题略有不同，所以，这个看似简单的任务却略显复杂了。其实这里只不过通常主题目录下的footer.php变成了templates文件夹下的end.php，所以，我们只需修改这个文件即可。

修改左下角的WP logo，只需把img文件夹下的wp-logo.png修改为你想要的图形；当然，你需要注意背景色的一致性。

修改WP logo的链接，只需将如上所述的end.php的第4行修改为：

<?php bloginfo(‘name’); ?>

这样，图像的链接就指向了你的站点首页；而且鼠标滑过图像时会显示你的网站描述信息。

修改页面底部你的站点名称为超链接形式，添加诸如备案信息等，则可以按照下面的方式。将end.php文件的第19、20行修改为：

echo $copyright; ?>
<?php bloginfo(‘name’); ?> 备案中 All rights reserved | Powered by Wordpress<?php

你应该能看懂上面的这些代码，如做改动，只需适当修改即可。需要提醒的一点是，任一个WordPress博客都应在站点底部保留WordPress的链接，请你不要忽略这点。至于其余的，比如将主题作者的链接修改为在新窗口中打开，则更是容易的事情了，在此，月夜也不再赘言。

10.修改主题中的一些文字

在iNove主题中，打开一个单独的页面之后，题目上方显示的是如“首页>”之类的链接信息，我们可以根据自己的喜好将其修改为如“月夜>”之类的形式。当然，我们也能修改页面中很多其他地方的文字，比如底部的“版权所有”、“置顶”等等。

首先打开主题文件夹下的languages文件夹，将其中的zh_CN.MO删除，然后用除记事本之外的其他程序，如Notepad2等打开，编辑相应的信息，如：

11.其他的一些说明

修改iNove主题的头部黑色长条，可以直接修改主题文件夹下img文件夹中的header_footer.jpeg图像的上半部分。

最后，月夜特别说明一点，在修改上面的所有代码时，你都要保证代码文件的编码方式是UTF-8，否则，如果出现汉字，则会显示为乱码！
