雷锋日随想

Posted on 2009-03-05 | In 时讯品评

今天沈阳的天气让人有些郁闷，中午时飘起了大雪，纷纷扬扬，似乎是 月夜 一生当中所见到过的最大的一次降雪。大雪毫不留情的倾泻了下来，搞的天地间雾气蒙蒙，给人一种很压抑的感觉。但所幸，大雪只下了不大一会儿，再后来就转小了；而且更所幸的是，气温不低，雪花刚落到地面就融化了，我们不需再为扫雪的事情发愁，再也不用怕像上次降雪后要扫雪扫到手抽筋。 月夜的Google AdSense帐号在点击党们的努力下，今天下午5时被Google封杀，光荣的殉职了。里面有我开博几个月的积蓄，然而，这一切都随风去了。月夜再也无法指望通过Google AdSense广告的微薄收入去糊网站的域名和空间费用的口。然而，也无非如此，不过是没了那一点收入，但随之而来的呢？却是月夜从浮躁心境中的解脱，却是月夜从终日被点击党们恶搞的烦恼中的解脱。 今天是雷锋日，我想很多朋友也都和月夜一样记住了这个特别的日子。 诚然，在目前的社会环境下，尤其是网络环境下，很多网友都很无奈或者是郁闷，以至于对很多东西都失去了希望，变得有些愤青。在现实中会有很多抱怨，在网上偶尔会“大开杀戒”的发泄一番，比如说说脏话，骂骂人，等等。 但正如我们所知道的那样，即使是一件没有任何可取之处的东西，也自有它的亮点。比如，我们就发现了雷锋。 即使你恨透了这个社会，即使你恨透了这个地方，但它还不是诞生了伟大如雷锋的一群人？雷锋不是中国的，雷锋是整个世界的！雷锋不短暂属于某个政府，而长久属于整个人类！即使某天沧桑尽无，雷锋终究还会是雷锋，我们一样还会继续学习着我们的雷锋，唱着我们熟悉的“学习雷锋好榜样”，在雷锋日或者雷锋月里洗剂我们的心灵，做那些让我们共同感动的事情！

从倪萍的封杀“山寨”说开去

Posted on 2009-03-05 | In 时讯品评

照例，每年两会热开时，电视、互联网上都少不了明星们那熟悉的身影和脸庞。今年首先跳出来的是倪萍。其实个人对倪萍的印象挺好，觉得人家起码也是一忠厚老大妈的形象吧；去年地震赈灾，人家还大大方方地捐了100万呢，对一个过气明星，实在是相当不易了。 然而这次，对倪萍提出的立法封杀“山寨”以及反对“山寨”文化和“山寨现象”的提案，我却不敢贸然苟同。 正如一位名人所言，这世界因为其多姿才显得美好；缺少了千变万化，何来美感？我记得倪大妈曾经演过一部非常感人的电影——《美丽的大脚》，个人觉得如果按照很多人的观点来看，里面的倪萍就是一个标标准准的山寨教师！但数亿电影观众是否因此就要求大家封杀过倪萍呢？其实，这个世界是需要宽容的，抱着宽容的心看天、看地、看世间万物，你会发现，原来世界竟如此美好。相反，如果我们像更年期的大妈一样，情绪激动，忿忿不平，说不定还真会写出如此的提案。 按照倪萍大妈的观点，这世上就应该只存在正规军，山寨就是没有爹妈的孩子，自然是要被他们扼杀。电视台只应存在CCTV，什么辽宁卫视、湖南卫视等等就应该从地球上被摸去；至于其他的一些名不见经传的小小TV就更不应有其生存之地。我冒昧的问一句，如果CCTV和ABC、NBC站在一起，那谁又是山寨呢？我还记得荷兰貌似是世界上第一个资本主义国家，那荷兰以后建立的所有资本主义制度政府，如英国、美国，是否都是山寨？是否应该被立法封杀，从这个星球上摸去？ 倪萍大妈说，“山寨文化”假的多、复制的多、盗版的多，很多都是盗版、造假。但我想提醒倪萍在递交提案之前，请先弄清自己提案的内容，搞懂什么是真正的“山寨”，搞懂什么是盗版、造假，当然，更需要搞懂群众为什么呼唤“山寨”，“山寨”为什么在当今社会如此风行！ 诚然，倪萍是CCTV走出来的正规军，人家肯定发自内心对“山寨”有一些不满和鄙视，如此的表达一些感情也自在情理之中。但诚如倪大妈自己所言，“名气是观众给的”，说话时还是对观众负责些比较好。

Posted on 2009-03-04 | In 网站相关

在前面的文章 iNove主题本地化的几点经验(一) 中，月夜 为您介绍了自己在将iNove主题本地化时的一些经验，比如拓宽页面、修改字号、修改标题样式、添加postviews插件等等。本文，月夜将接着上文中的话题，继续为您介绍我在iNove主题本地化中的其他一些经验。

与前文相同，月夜使用的iNove主题版本是1.2.3。而且，在我们修改主题文件时，所提及的文件以及文件信息依然指未修改前的绝对位置，而不指修改中的相对位置。

6.修改页面顶端代表首页的“小屋”图标

不知道诸位的感想如何，月夜对那个图标不是太满意，而且还太小，感觉很不醒目。因此月夜想换掉那个图标。

首先找到主题文件夹下img文件夹中的menu.gif图像，使用适当的工具将其上的“小屋”图标换成你想要的图形或文字，并大致估算一下新图标所占据的宽度。然后，我们打开style.css文件，找到其中第225行，将其修改为：

width:80px;

不过，你需要根据自己图标的宽度来适当修改此处的数值；当然，你也可以一点一点的修改，直到你的首页图标位置另你满意为止。

8.在文章页添加上版权信息

在每一篇文章中都自动添加上一些版权信息，以提示网友在转载时注明出处。我们可以这样在iNove主题中实现。

首先打开style.css文件，在其中第485行之后，添加如下的三行代码：

.reship{ float:left; width:99%; margin-bottom:20px; font-size:12px; }
.reship p{ padding:0px 10px 0px 0px; text-align:right; color:#999999; border-right:1px dashed #BBBBBB; font-size:12px; }
.reship a:link, .reship a:visited, .reship a:hover{ color:#999999; text-decoration:underline;}

然后打开主题文件夹下的single.php文件，找到其中第27行，在其后面添加如下代码：

转载原创文章请注明：文章转载自：<?php bloginfo(‘name’); ?> [<?php echo get_settings(‘home’); ?>]

本文标题：<?php the_title(); ?>

本文地址：<?php the_permalink(); ?>



这样，我们就成功在博客每一篇文章的后面添加上了版权信息。当然，你也可以修改上面的代码来添加上属于自己的个性版权信息。

iNove主题本地化的几点经验(一)

Posted on 2009-03-04 | In 网站相关

如前文 月夜WordPress主题稍作修改 所述，月夜 最近在建立一个新的专业博客，而为了省事儿，在新的博客上使用了现在比较流行、功能又比较强大的iNove主题。尽管主题已经比较强大，但月夜还是按照习惯在使用前将其进行了简单的本地化。因为现在使用iNove主题的朋友比较多，月夜将自己的一点经验分享出来，希望能给后来者带一点方便。

在进行下面的叙述前，月夜先说明一下，我们使用的iNove主题版本是1.2.3。还需说明的一点是，在下面的修改中，所提及的文件以及文件信息均以未修改前的绝对位置为准，而不指修改中的相对位置。

1.拓宽整个页面

iNove主题本来的页面宽度只有936px，在当今主流分辨率均超过1280x1024的屏幕上显得比较浪费空间；而且文章页面的宽度太窄就没法插入相对较大的图片。因此，月夜就将主体稍微进行了一些拓展，扩展到了980px；与此相应，修改左侧文章页的宽度为649px（此处是为了给侧边栏的分界线留下空隙），侧边栏保持300px不变。

修改的主题文件夹下style.css文件的代码如下：

修改第151行为：width:980px;
修改第309行为：width:649px;

当然，与此相应，就要修改主题文件夹中img文件夹下的header_footer.jpeg文件的宽度，将其修改到与新的主体宽度对齐；而其中的light.gif，如果要求不高，则可以不做修改（当然，你也可以进行修改以使其更完美）。

2.修改文章字体字号

iNove主题中文章字体的字号实在太小，给网友访问阅读文章带来一些不便，所以下面我们将其字体稍微修改加大一些。

同样是修改style.css文件：

修改第24行为：font-size:14px;
修改第496行为：font-size:14px;
修改第611行为：font-size:14px;

但因为其中如首页、存档页等页面上文章标题下方的阅读评论、文章发布日期等的字号没有太大影响，所以没有修改。

3.修改博客题目和描述样式

iNove主题的博客名和博客描述显示效果让很多人感觉不是太满意，毕竟很少有人愿意去做像MG12的那种题目效果。月夜试着稍作修改：

修改第162行为：padding:15px 30px 21px;
从第172行开始添加：font-size:32px;font-weight:bolder;
从第164行开始添加：#caption a { color:#FFF; font-size:24px; }
将第185-192行的模块注释掉（或者删掉）;

接着，我们打开主题文件夹下templates文件夹下面的header.php文件，将其中的第5行修改为：

<?php bloginfo(‘description’); ?>

当然，你也可以适当修改如上的描述文字指向的页面（月夜指向的是关于页面）。

4.修改页面的title

很多人有修改新主题title以使得更改主题前后网页页面的title保持一致。iNove主题的title在templates文件夹下start.php文件的第42行，你可以根据自己的需要将其修改为自己想要的形式。月夜修改的代码如下：

<br><?php if (is_home() ) { ?><br><?php } else {?><br><?php wp_title(‘’); ?> | <?php } ?><br><?php bloginfo(‘name’); ?> - <?php bloginfo(‘description’); ?><br>

5.添加postviews插件

wp-postviews插件能够记录并显示你的博文被阅读的次数，以方便访客查看你最受欢迎的文章，很多朋友都在使用，当然，月夜也不例外。下面我们说说如何在iNove主题上添加postviews显示的阅读次数。

首先在style.css文件中第341行插入

.post .postviews,

接着在文件的第377行插入

.post .postviews {
background-position:0 -16px;
float:left;
}

在主题文件夹下的index.php文件第19行之后插入：

<?php if(function_exists(‘the_views’)) { the_views(); } ?>

这两天 月夜 忙着新建另一个WordPress博客（为了防止点击党们的恶意破坏，月夜不会放出此博地址。谢谢朋友们的关注），为了省事，就选择了目前十分流行、功能十分强大的iNove主题。 照例，月夜在使用一款新的主题时总是会做些改动，即将主题本地化，iNove也不例外。当然，因为iNove主题比较强大，而且，因为是做专业博客，也没有太多时间专注主题，所以月夜并不想修改太多，因此改动的地方很少。但在改动过程中，还是有一些经验；而且因为目前使用iNove主题的博友很多，所以，月夜准备在接下来的文章中和大家分享一下iNove主题本地化的经验。但在此文，月夜只是简单叙述一下自己对现在使用的这款WordPress主题所作的一些小小改动。 1.修改了超链接的样式 月夜使用的这款菠菜主题，开始时超链接的样式和普通字符没有什么区别，使用起来很不方便；而且鼠标滑过时的颜色变成了暗红色，看起来也不太美观。后来，月夜进行了简单修正，统一修改成了超链接最常用的样式——使用了正蓝色。不过正蓝色的超链接在文章中看起来又太过显眼，破坏了整个页面的美观性。 这次看到iNove主题，发现里面的超链接样式看起来很不错。就修改成了它的样式，但为了醒目，月夜还是保留了鼠标划过时的正蓝色。 2.修改了related posts插件的显示样式 一直以来月夜都感觉博友们的相关文章插件用起来很爽，但一方面没有时间，另一方面月夜又天生有些惰性，所以一直以来也没有添加。 前几天，月夜刚好空闲，就顺势下载了插件，安装了上去。但无比郁闷的是，插件显示的相关文章看起来极不美观，让人觉得十分不爽。不过，月夜的水平有限，因为wp_related_posts插件只有一个PHP文件，我没有找到相应的控制显示样式的CSS文件。无奈，只有作罢，任其丑陋地显示在月夜的博客上。 刚好在iNove中看到了这个插件的显示效果，非常好看；在style.css文件中又找到了控制显示样式的代码，心想，原来可以这样做。于是，月夜就把iNove主题style.css文件中第949-993行的代码复制了出来：

/* related posts START / / when related posts with title / #related_posts { padding-bottom:10px; } #related_posts h3 { background:#E3E4E6 url(img/box.gif) 0 0 no-repeat; font-size:12px; padding:6px 12px; font-family:Verdana,”BitStream vera Sans”,Arial,Helvetica,Sans-serif; letter-spacing:0em; border-bottom:1px solid #CCC; } #related_posts ul { background:#F4F5F7 url(img/box.gif) 100% 100% no-repeat; padding:3px 12px 8px; } #related_posts ul li { list-style:none; padding:3px 5px; border-bottom:1px solid #E3E4E6; line-height:18px; } #related_posts .count { font-size:10px; } #related_posts .excerpt { color:#999; font-size:11px; display:block; } / related posts END */

将其放到月夜使用的菠菜主题style.css文件中相应的位置(注意需要根据自己的情况适当修改其中的url(img/box.gif)一句，以使图片路径正确)。保存之后，将iNove主题下img文件夹中box.gif文件上传到月夜的主题文件夹下的图片文件夹中，刷新页面，月夜终于告别了那可恶的相关文章样式，迎来了如今的美好面貌。

月夜备案成功

Posted on 2009-03-02 | In 网站相关

尽管这两天一直被一群点击党们不断恶搞，什么恶意刷流量，什么在cnBeta上恶意留言侮辱 月夜，不一而足，花样层出不穷，不过月夜也慢慢习惯了这种折磨。其实对待他们的最好方式就是置之不理，由他们去，爱怎么地就怎么地。只要好好的写自己的博，其他的管那么多干嘛呢？慢慢的努力，踏踏实实的去写，终归会有成功的消息，也肯定会有不少收获。 这不，今天下午，月夜无聊时去信产部备案管理系统闲逛，无意间发现月夜（https://www.yueye.org）已经通过了备案！信息提示上已经变成了“现处备案阶段为： 审核通过”！ 我真的好激动，看看提示一栏“尊敬的用户，您于2008-11-26进行了首次报备”，没错，从开始报备到今天通过，耗了3个多月！回想开始在备案管理系统的网站上注册账户时的郁闷；回想第一次报备时的憧憬；回想过年后来到所里看到的您的博客网站需要在省通信管理局进行专项报备；回想第二次将服务类型删去抱着死马当活马医的态度重新提交…回想这一切，月夜都觉得十分不爽，觉得无比郁闷，觉得真是莫名其妙。 但在数月郁闷之后，月夜终于备案通过，获得了自己的备案号——辽ICP备09002157号！月夜从此告别了游击生活，终于可以抬起头来光明正大的做站了，终于成了得到政府认可的网站正规军！ 其他的就不多说什么了，分享一点经验。博客备案并不是一件容易的事儿，所以在填写“网站内容类型”和“网站服务内容”时，不要填写博客，也不要选择选项中的博客，两项都留空即可。只是要注意，其他的信息尽量还是填写上自己的真实资料，因为工作人员有可能拨打你的电话或手机进行验证（月夜就接到了这样的电话）；至于姓名和身份证号之类的信息，那自然就更需要准确无误了。 OK，到此结束。月夜在此恭祝诸位朋友都能备案顺利，及早拿到属于自己的备案/许可证号！

月夜盘点2009年2月中国互联网事件

Posted on 2009-03-01 | In 时讯品评

网络反低俗活动的持续推进，春晚《不差钱》衍生语的流行，“躲猫猫”事件的朴树迷离，牛奶行业再爆丑闻，瑞星事件浮出水面，2月注定会在2009年的中国互联网上写下浓墨重彩的一笔！ 1. 抗旱救灾

自去年11月开始，我国北方大部地区滴水未降，小麦能农作物面临着极其严峻的旱情。时过春节，一场抗旱救灾活动在北方全面拉开。从河南、安徽到山东、河北，继去年年初大战雪冻灾害之后，又开始了一场新的战争。2月底，由于人工降雨以及气候的影响，抗旱应急响应解除。

2. 整治互联网低俗之风专项行动持续进行

进入二月，整治互联网低俗之风专项行动在互联网上持续深入进行，又先后通报并关闭了大量低俗网站。甚至传出了将关闭所有未备案网站的消息，看来这次网络反低俗活动还将继续推进。

3. “躲猫猫”事件

2月12日，盗伐林木被刑拘的云南玉溪北城镇24岁男子李荞明在看守所度过11天后因重伤入院，随后，于12日凌晨不治身亡。警察称其与狱友玩“躲猫猫”游戏时，遭到狱友踢打并不小心撞到墙壁而导致受重伤。新闻被各大网站转载之后，“躲猫猫”立即成为继“俯卧撑”之后的网络流行语。

然而，事情还没有结束。2月19日，为了解决“躲猫猫”事件带来的危机，云南省委宣传部决定成立由网友组成的网民调查团前往晋宁看守所实地调查真相。2月20日，网民调查团发布调查报告，但并未给出令人满意的答复。2月27日，云南省政府新闻办召开新闻发布会，就云南籍男子李荞明在昆明市晋宁县看守所死亡一事进行通报，李荞明系看守所内牢头狱霸以玩游戏为名，殴打致死。至此，经过网友们的不懈努力之后，“躲猫猫”事件最终真相大白。

4. 蒙牛特仑苏OMP事件

2月2日，国家质检总局指中国乳业巨头蒙牛集团的高端品牌“特仑苏”违法添加了名为OMP(造骨牛奶蛋白)及IGF-1（类胰岛素生长因子）的物质，而我国尚未明确OMP的安全性，IGF-1也未列入食品添加剂使用标准。2月11日，此函被媒体获悉并公布。此次官方介入源于质检总局接到对蒙牛特仑苏产品的举报，是官方首次对蒙牛的“OMP”进行表态。

5. 丁磊养猪

2月16日，从不同渠道传出的消息得知，网易CEO丁磊计划在浙江湖州、金华一带搞养猪场，养殖规模10000头。网易方面的消息称，丁磊以高科技创业成功者身份介入养殖业，希望通过这一行为打破行业观念、产业界限等思维定势；丁磊将商业、投资与公益、民生有机结合，给处于萧条期中茫然不知所措的创业者们提供新思维。

6. 瑞星事件

2月17日，《科技日报》刊发了题为“一项重大原始创新何以大难不死——北京东方微点公司起死回生始末”的特稿，披露了瑞星公司行贿执法人员，联手制造了微点公司的假案，致使后者产品拖延三年方才上市，以及公司副总被警方逮捕关押11个月。《北京晚报》记者采访了相关各方，力图还原这桩堪称中国计算机杀毒业界最大的丑闻。瑞星公司被推上风头浪尖，其生存面临着空前严重的危机。但事件还远远没有结束，时至今日，事件双方还一直在不停的攻击对方，并将事件诉诸法律。

7. 百度百科“十大神秘饮食”

2月20日，继百度百科“十大神兽”之后，百度百科再遭戏弄，被网友恶搞出了“十大神秘饮食”，这些神秘饮食分别为：墙煎饭(强 奸 犯，饭)、干泥酿(干 你 娘，酒)、菊鲍(菊爆，鲍鱼)、命羹(命 根 子，粥)、春水糖(春 水 淌，糖果)、盐鸠参(研究生，参汤)、厨女馍(处 女 膜，馍)、糕丸(睾 丸，肉丸)、褒二奶(包 二 奶，饮料)和奥特馒(奥特曼，馒头)。
