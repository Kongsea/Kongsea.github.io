网站建设全过程（使用cPanel X）详细讲解（三）

Posted on 2008-12-01 | In 网站相关

在前面的文章中我们已经介绍了网站建设的前期准备工作，包括注册域名、申请空间以及对空间的初步熟悉，当然，我们也已经准备好了用于网站建设的邮箱和MySQL数据库等等。 今天，51Blog接着前面的内容为您继续讲述有关网站建设的全过程。

四、通过FTP上传WordPress程序

事实上这些步骤我们在前文网站建设全过程（使用Vista Panel）中已经为大家讲述过了，在此我们只简单的一笔带过，如果您想了解更详细的内容，请点此阅读。

1.下载WordPress简体中文版程序

点此直接下载程序：WordPress 简体中文版 。下载完成以后，解压，并取出解压后的文件夹中的WordPress文件夹以供我们使用。

2.配置FTP工具

这部分内容我们在此略去，如果您还不清楚相应的配置，那么请点此阅读详情。

3.在FTP工具中建立我们的FTP站点

首先选择并点击“文件”-“新建”菜单下的“FTP站点”选项，软件将弹出站点属性对话框，如下图所示：

在弹出的站点属性对话框中输入如上图所示的内容，即：

主机标签：***

主机地址：kongsea.net23.net

用户名：a***

密码：**

然后点击连接按钮，FTP软件即开始连接我们的FTP站点，等待片刻，连接完成的页面如下图所示：

只需注意如上图所示的public_html文件夹即可，其他内容可以忽略，我们的网站程序都要上传到该文件夹中。双击该文件夹，我们可以看到其中的内容如下图所示：

其中的default.php是系统生成的站点默认页，我们可以直接将其删除，然后来上传我们自己刚刚下载的WordPress程序。

4.上传脚本程序

我们首先还是要打开WordPress文件夹，来修改其中的wp-config.php文件中的配置。如网站建设全过程（使用cPanel X）详细讲解（二）一文中所述，关于我们的数据库的信息如下：

$mysql_host = “mysql4.000webhost.com”; $mysql_database = “a3202806_wp”; $mysql_user = “a3202806_i”; $mysql_password = “**“;

打开解压后的WordPress文件夹，找到wp-config.php文件，打开后（需注意，不可使用记事本打开修改）找到其中的第2行到第6行，根据上面的数据库信息将其修改为如下内容：

// ** MySQL 设置 // define(‘DB_NAME’, ‘a3202806_wp’); // 数据库名称 define(‘DB_USER’, ‘a3202806_i’); // MySQL 用户名 define(‘DB_PASSWORD’, ‘**‘); // …和 密码 define(‘DB_HOST’, ‘mysql4.000webhost.com); // 99% 的情况下，您不需要修改这一项

然后保存，并可以将WordPress文件夹中的wp-config-sample.php文件删除。 将前面我们修改过的文件连同整个WordPress文件夹中的内容按照网站建设全过程详细讲解(四)一文中“4.上传脚本程序”方法上传到public_html文件夹中。 上传成功后的FTP软件一瞥如下图所示：

五、安装WordPress程序，建立网站

上传完毕WordPress程序之后，我们即可安装程序了。在浏览器地址栏中输入形如“http://kongsea.net23.net/wp-admin/install.php”的地址，然后回车，系统即跳出安装界面，如下图所示：

在其中输入相应信息，点击“安装WordPress”按钮，进入下一步，即可看到WordPress安装成功的提示，如下图所示：

将其中的密码复制下来，并记住用户名，然后点击登录按钮，即可出现系统后台登录对话框，如下图：

在登录对话框中如上图所示输入用户名和密码，然后点击登录，即可成功登录到网站后台。登录成功后的后台一瞥如下图所示：

在后台你可以进行WordPress的各种设置，也可以发表文章，添加页面等等。不过记住，你的第一件事应该是修改自己的账户密码！修改完成密码之后，是不是想看一下自己辛苦劳作的成功啊？点击形如http://kongsea.net23.net的地址访问你的WordPress博客吧。

好了，今天，我们关于网站建设全过程（使用cPanel X）的讲解就先说到这里，在接下来的文章中，51Blog将带您去详细配置自己的WordPress站点，包括更换主题，修改设置等等。更多精彩内容，请继续关注51Blog。
