WordPress安装完成后需要做的几件事（二）

Posted on 2008-12-03 | In 网站相关

在前面的文章WordPress安装完成后需要做的几件事（一）中，51Blog带您初步了解了一些在WordPress博客安装完成之后要做的事情：WordPress博客的基本设置。本文中，51Blog将带您继续了解在WordPress博客安装完成之后要做的工作，本文我们主要来介绍一下WordPress博客的主题更换以及启用相应的插件等等。

一、更换WordPress的主题

WordPress的一大吸引人之处就在于其可以自由更换主题，得到自己想要的或者是喜欢的样式和风格；WordPress安装完成之后的默认风格肯定不是大家想要的，51Blog使用的是基于蔬菜博的主题而自己稍作修改的样式，你可以直接查看效果。 当然，事实上现在WordPress的主题已经多的数不胜数了，你可以自由的选择自己喜欢的样式来安装使用。你可以点击这些链接：www.wpthemesfree.com、www.wpthemespot.com等等来寻找自己喜欢的主题。当然你也可以使用Google来搜索相应的主题站点去搜寻自己喜欢的主题。然而，因为WordPress主题越来越多，所以从中选择一个自己喜欢的样式也并不是一件容易的事情。不过没关系，有人提供了在线订制属于自己的模板的网站，你可以点此订制自己喜欢的主题。 搜索到自己喜欢的主题之后，直接将其下载，我们就可以进行接下来的操作了。 下面，51Blog以将自己的WordPress小站http://kongsea.net23.net/的主题修改为蔬菜博主题为例，来为大家介绍一下如何更换自己的WordPress博客主题。

首先将下载下来的主题文件解压，请从中取出主题文件夹（如果是多层文件夹，只要取出单个文件的上一层即可；当然，你也可以将文件夹最里面的多个文件直接复制到一个文件夹中使用）。整理好主题文件夹之后，接下来我们就可以将其上传到站点目录中了。需要注意的是，我们需要将主题文件夹上传到站点根目录中的wp-content/themes/文件夹下，上传完成之后的FTP客户端界面如下图所示：

可以看到，上传的BBtheme2主题文件夹是放在了/public_html/wp-content/themes文件夹之下的，而且，我也自作主张将WordPress默认的classic主题给删除了，只保留了一个default主题。上传完毕之后，我们就可以进行接下来的操作了。 上传完成之后，我们进入到WordPress博客的后台，点击设计菜单下的主题子菜单，可以看到如下图所示的页面：

我们可以在这里看到当前使用的主题，还可以在下侧看到所有我们已经成功安装（虽然没有使用）的主题。如果想更换主题，只需点击相应的主题缩览图，比如此处，我们点击左侧的蔬菜博主题缩览图，系统即弹出预览界面，如下图所示：

如果你对这个主题的效果感到比较满意，直接点击预览图右上角的“启用“BBtheme V2.0””即可启用相应的主题，启用成功之后，主题菜单有如下图所示的提示：

直接点击上图中的访问网站超链接，即可查看更换主题之后的WordPress博客的效果。

当然，如果你对相应的主题感觉不是十分满意，你可以继续挑选相应的主题；或者，如果你对CSS和PHP等稍有研究的话，可以自己对主题进行适当的修改。比如目前51Blog使用的主题就是在蔬菜博主题的基础上进行了多处修改而得到的结果，当然，51Blog还在对其进行修改，以期达到心中的完美。

二、安装WordPress插件

为了增强WordPress博客的功能或者提高WordPress的易用性，很多PHP高手自愿为WordPress开发了很多插件，作为一个WordPress博客使用者，我们可以直接把这些插件拿来使用。 像WordPress主题一样，你可以进入相应的插件站点来查找自己需要的插件，或者通过Google来搜索相应的插件；当然，你也可以使用自己需要的功能来直接搜索相应的插件，比如你要在自己的博客上放置Google AdSense广告，就可以直接搜索“WordPress AdSense 插件”，从而来找到适于自己的插件。 51Blog目前使用的插件主要有这些：

1.Google XML Sitemaps（用于生成站点地图以便于搜索引擎收录） 2.WP-UserOnline（用来查看在线用户数量） 3.AdSense-Deluxe（管理放置AdSense广告） 4.WordPress Database Backup（备份WordPress数据库） 5.蔬菜博主题需要的WP-PageNavi、WP-PostViews以及Vote It Up三个插件

当然，你可以根据自己的需要来选择相应的插件。 下面，51Blog以安装设置Google XML Sitemaps插件为例来简单讲述一下如何安装使用WordPress插件。

首先将下载的插件文件夹解压整理好（如果是多层文件夹，只要取出单个文件的上一层即可；当然，你也可以将文件夹最里面的多个文件直接复制到一个文件夹中使用）。然后将解压整理好的插件文件夹上传到WordPress站点根目录的/public_html/wp-content/pluguns文件夹之下，上传好的FTP客户端如下图所示：

上传完毕之后，我们进入到WordPress博客后台，点击右侧的插件菜单，可以在禁用的插件中找到我们刚刚上传的Google XML Sitemaps插件，如下图所示：

点击插件右侧的“启用”超链接，即可启用该插件，启用后，我们可以在当前启用的插件中看到启用的插件，如下图：

插件至此已经安装完成。然后，我们来对插件进行一些设置。点击“设置”菜单，在菜单下可以发现我们刚刚启用的Google XML Sitemaps插件的设置菜单“XML-Sitemap”，在下面的选项中，我们可以看到如下图所示的内容：

可以看到，需要创建两个文件：.xml文件和.gz文件，而继续往下翻看选项部分，可以看到.xml文件的地址是http://kongsea.net23.net/sitemap.xml，路径为/public_html/sitemap.xml，所以，我们需要在/public_html/文件夹下创建两个可读写的文件以供该插件使用，两个文件分别为：sitemap.xml和sitemap.xml.gz。当然，你可以本地使用记事本新建文件之后保存为这样的名称，然后将其上传到/public_html/根目录下并修改权限属性为666即可。 至此，我们已经创建好了相应的文件，然后点击如下图所示的插件选项中的点击这里超链接来创建站点的XML地图文件：

当然，事实上该插件还有很多设置选项，但那些不属于本文的内容，故在这里，51Blog也不再赘述了，你可以通过搜索引擎来查找相应的教程。当然了，如果你留言提出问题的话，51Blog还是很乐意为你解答的。
