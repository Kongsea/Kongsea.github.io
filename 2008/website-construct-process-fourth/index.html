网站建设全过程详细讲解(四)

Posted on 2008-11-21 | In 网站相关

接着前面的网站建设全过程详细讲解(一) 、网站建设全过程详细讲解(二) 和网站建设全过程详细讲解(三) ，51blog继续为您详细讲解网站建设的全过程。

五、自己上传脚本程序建设网站

在前面的网站建设全过程详细讲解(三)一文中，51blog为大家详细讲述了如何使用免费空间提供商提供的Fantastico脚本程序建设自己的网站。但空间商提供的脚本程序几乎全部是英文程序，不利于我们的使用；而且空间商提供的脚本大多不是最新版本，有一定的安全隐患。所以，51blog不推荐使用那种方式建立网站，而推荐自己下载相应的简体中文版最新程序，然后上传，并创建自己的站点。

1.下载安装程序

首先，51blog来为大家提供一个WordPress简体中文版的下载地址：WordPress简体中文版 。 点击该地址右键另存为可直接下载，或者使用下载工具进行下载，不到2M的程序，下载十分容易。 下载完毕以后，解压，取出解压后的文件夹中的WordPress文件夹以供我们使用。

2.配置FTP工具

需要注意的是，很多脚本程序的上下载过程中都需要采用二进制方式，所以，我们需要对FTP工具进行适当的配置。在这里，51blog以CuteFTP为例来简单介绍一下如何配置FTP软件。 如果您还没有下载安装CuteFTP软件，可以点此搜索该软件，然后下载安装。 安装好软件以后，我们来进行配置。打开CuteFTP软件，如果您的CuteFTP软件是简体中文版，则可直接跳过这一步骤。如果您的软件以英文方式显示，则请进行简单配置，将其修改为中文。 选择Tools菜单下的Global Options子菜单，打开Global Options对话框，如下图所示：

按照上图，选择语言选项，然后在右侧将语言从Default.lng改为Chinese.lng，并点击OK按钮。接着重启CuteFTP软件，即可将软件的显示语言修改为简体中文。 修改完成以后，我们接着进行下面的设置。选择工具菜单下的全局选项子菜单，打开全局选项对话框，如下图所示：

在左侧栏中选择传输，然后在右侧的传输方式中选择“二进制”，点击应用按钮，将传输模式修改为二进制模式。然后再点击左侧栏中的ASCII类型，如下图所示：

检查右侧的下拉列表中是否有PHP类型，如果没有，跳过此步；如果有的话，选中，并点击删除按钮，将其从列表中删除。然后点击确定按钮，即完成了CuteFTP软件的设置。 设置完CuteFTP软件以后，我们来建立ZobyHost的站点。选择并点击“文件”菜单下“新建”菜单下的“FTP站点”选项，软件将弹出站点属性对话框，如下图所示：

在如上图所示的对话框中，依次输入您为站点加的标签，FTP服务器的主机地址ftp.zobyhost.com，用户名zoby_***和密码，然后点击连接，可以看到软件已经开始连接服务器。稍等片刻，如果没有什么意外的话，即可连接成功。连接成功后的软件界面一部分如下图所示：

可以看到，目前在我们的空间下面有两个文件夹，一个是htdocs，这是空间商为我们提供的二级域名（如http://deeepsea.zobyhost.com）所使用的文件夹，打开该文件夹可以看到我们在前面使用空间商提供的一键安装程序建成的网站的文件；另一个是***.co.cc文件夹，这个是我们自己绑定的域名所使用的文件夹，打开该文件夹可以在其中发现一个htdocs文件夹，这个文件夹是我们保存网站程序的地方，一会儿我们上传程序，就是要上传到该文件夹下。 这里需要说明一点的是，也有很多空间商要求程序上传到public_html文件夹中，对不同的空间商，需要稍加注意。 好了，至此，我们已经把FTP软件配置好了。

3.创建用于网站建设的MySQL数据库

配置好FTP软件以后，我们还需要为网站建设再做一项准备工作：创建MySQL数据库。 事实上在前面网站建设全过程详细讲解(二)一文中，我们已经详细讲解了如何在ZobyHost空间中创建MySQL数据库，所以此处我们就不再详细讲述。只要朋友们自行创建一个名为zoby_*_wordpress**（记住创建时方框内填入WordPress）的数据库就可以了。 创建好相应的数据库以后，我们再来做一下配置。打开上面刚刚下载并解压缩的WordPress文件夹，在其中找到wp-config.php文件，使用写字板打开，在第2行到第6行找到如下所示的部分：

// ** MySQL 设置 ** // define(‘DB_NAME’, ‘demo-wp’); // 数据库名称 define(‘DB_USER’, ‘test’); // MySQL 用户名 define(‘DB_PASSWORD’, ‘test’); // …和 密码 define(‘DB_HOST’, ‘localhost’); // 99% 的情况下，您不需要修改这一项

将其修改为：

// ** MySQL 设置 // define(‘DB_NAME’, ‘zoby_*******_wordpress’); // 数据库名称 define(‘DB_USER’, ‘zoby_*******‘); // MySQL 用户名 define(‘DB_PASSWORD’, ‘**‘); // …和 密码 define(‘DB_HOST’, ‘sql200.zobyhost.com’); // 99% 的情况下，您不需要修改这一项

然后直接将文件保存即可。事实上以上各项设置的意义十分明显，但为了防止出错，在这里，51blog还要为大家解释一下各项的意义，首先第2行是数据库名称，就是我们刚刚创建的数据库的名称；第3、4行是数据库的用户名和密码，对ZobyHost空间来说，和控制面板用户名和密码相同；第5行就是数据库的主机名，我们在前文网站建设全过程详细讲解(二)和网站建设全过程详细讲解(三)中已经说明并使用到了这个东西，所以此处也不在赘述。 设置并保存完毕之后，接下来我们就可以上传脚本程序了。

4.上传脚本程序

首先使用CuteFTP软件打开刚刚建立的ZobyHost FTP站点，并将服务器端定位到***.co.cc/htdocs文件夹下，如下图所示：

然后将如上图左侧所示的我们电脑端刚刚下载并解压的WordPress文件夹中的全部文件选中并拖到右侧的服务器端文件夹中，FTP软件即开始上传文件，上传中的一瞬如下图所示：

稍等片刻，等待文件上传完毕，我们接着进行下面的操作。

5.安装脚本程序

上传完毕脚本程序之后，直接在地址栏内输入形如：http://***.co.cc/wp-admin/install.php的地址，然后回车访问，即可开始安装程序了。和前文网站建设全过程详细讲解(三)中使用空间商的脚本程序安装程序类似，首先我们可以看到如下图所示的页面：

在如上图所示的页面中输入博客名称和站长邮箱，然后直接点击安装WordPress按钮，即可开始安装过程。安装几乎不需要什么时间，系统将直接跳转到安装成功页面，如下图所示：

注意记住用户名和密码，然后点击登录按钮，系统跳转到登录网站后台页面，如下图所示：

在如图所示的登录框中输入用户名和密码，然后点击登录按钮，即可成功登录到后台。登录后的后台页面一瞥如下：

好了，至此，我们通过上传文件安装WordPress成功完成。你可以点击访问网站来查看网站的效果。当然你也可以点击撰写来开始写博。不过51blog特别提醒您一点，登录后台以后一定要记得首先修改一下自己的密码。 好了，今天的网站建设全过程讲解就说到这里，更多内容请继续关注网站建设全过程详细讲解(五)、网站建设全过程系列文章以及51blog。
