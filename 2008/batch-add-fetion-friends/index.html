批量添加飞信好友

Posted on 2008-11-12 | In 经验知识

昨天是传说中的光棍节，在此为兄弟姐妹们提供一个搜索并添加MM或GG的方法。 事实上我们大家都经历过使用QQ搜索添加好友的日子，但那早已是过去的事情了。而且就目前来说，使用QQ搜索好友有很大的缺点和不足。一方面，QQ用户众多，但虚假信息也较多，而且经常使用QQ的人可能会是网虫，这往往不是我们想要的；另一方面，QQ没有办法添加一个地方的好友，而且也不能批量添加。而使用飞信，则完全没有了这些缺点。飞信和手机相连，保证了信息的真实性；飞信可以批量添加好友；而最重要的一点是，我们使用飞信可以定向添加一个地方的好友，比如，你知道其中的一个手机号是某某大学的号码，那么你可以将尾号的最后几位适当修改，那么几乎可以保证，修改后的手机号也是该大学的号码。 闲话不提，下面我们开始批量添加飞信好友。这种批量添加的方法使用了飞信的导出导入好友列表功能。 首先，我们在飞信的窗口中右键点击，可以看到如下图所示的菜单：

首先，我们选择导出好友列表菜单导出好友列表，来查看一下飞信好友列表的格式，如下图所示：

看到了飞信好友列表的格式，我们就可以编写程序来生成类似的好友列表了。 C++程序代码如下图所示：

#include <iostream.h> #include <time.h> #include <stdlib.h> #include <fstream.h> long int PrintNumBasic=0, PrintNumBegin=0, PrintNumEnd=0; void main() { cout<<”请输入您想生成的手机号序列的前7位（如：1358888）:\n”; cin>>PrintNumBasic; cout<<”请输入您想生成的手机号序列后四位的起始数（输入数字，不要加前缀的0，如20）:\n”; cin>>PrintNumBegin; cout<<”请输入您想生成的手机号序列后四位的结束数（输入数字，不要加前缀的0，如99）:\n”; cin>>PrintNumEnd; ofstream outfile(“out.fcl”); outfile<<”<?xml version=\“1.0\“ encoding=\“gb2312\“ ?>\n”; outfile<<”\n”; srand(unsigned(time(NULL))); for(long int i=PrintNumBegin;i<=PrintNumEnd;++i) { outfile<<”“; outfile<<PrintNumBasic; if(i<10) outfile<<”000”; else if(i<100) outfile<<”00”; else if(i<1000) outfile<<”0”; outfile<<i; outfile<<”“; outfile<<”\n”; } outfile<<”“; }

将上面的代码在VC中编译连接生成相应的程序以供我们下面使用。当然你也可以点此直接下载编译好的程序。 将编译或者直接下载的程序复制到一个文件夹中，在这里我推荐你复制到根目录下（比如E盘下）。然后调出DOS命令行窗口，如下图所示（下例中程序名为MobileNumGen，放到了E盘根目录下）：

调出命令行后，输入e:并回车将命令行目录切换到E盘，然后输出程序名，如“MobileNumGen”回车，将开始执行程序。根据程序相应提示，输入相应的信息，程序就会在E盘根目录下生成飞信好友列表文件。生成的好友列表文件如下图所示：

然后我们在飞信中批量添加好友。首先还是在飞信主程序窗口中右击，从飞信弹出的快捷菜单中选择导入好友列表，飞信将弹出如下图所示的“导入联系人”对话框：

我们选择从包含联系人信息的文件导入选项，然后单击浏览，选择刚刚生成的飞信联系人文件，如下图所示：

选择好文件以后，直接点击下一步，在弹出的对话框中选择你想要添加的好友，或者直接点击全选，接着点击下一步，即可完成飞信好友批量添加了。 注：文中所述方法不可用于非法用途，因此造成的任何后果均由使用人负责，月夜博客不承担任何责任。
