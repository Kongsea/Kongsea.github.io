安装Win 7后的几件事儿

Posted on 2012-03-18 | In 经验知识

前段时间买来笔记本之后，装系统、弄软件，为将新的电脑收拾到自己用起来顺心自如，这期间辛辛苦苦折腾了一大段时间。而今，终于已经基本上算是把电脑弄好了，但这中间事无巨细的琐事，似乎应该做一个梳理，记录下来也可以供自己以及后来者做一个参考。

Windows 7操作系统

虽然题目是装好系统之后做的事儿，但首先还是以安装系统的过程来做本文的一个引子吧。在而今的形势下，安装系统自然是Windows 7，不过我没有Win7系统盘，而在笔记本安装好的Linux基础上安装Windows 7，虽然电脑上有光驱，可似乎使用通过U盘启动的Win PE进行安装也会是最好的方式。 目前的Win PE种类繁多，不过综合考虑一下，似乎复出的老毛桃推出的工具箱是最好的方式。而至于是安装32位的x86还是安装64位的x64则完全可以根据计算机硬件和自己的需求灵活选择，如果拥有4G以上的内存，且对兼容性问题没有苛刻的要求（事实上如今兼容性问题基本上不存在），则可以选择64位的系统；如果内存在4G以下，且对兼容性要求较为严格，则最好安装32位的系统。安装完成之后的激活也有多种方式，有刷BIOS的硬激活，也有通过软件方式的灵活的激活方法。

虚拟光驱

安装完Windows 7系统之后，自然是一些自己比较常用的大型软件。不过在安装这些大型软件之前，最好先装上一个虚拟光驱，会给后来的安装提供不少便利。月夜安装的虚拟光驱软件是DTLite，也就是大名鼎鼎的Daemon Tools Lite。事实上因为此款虚拟光驱已经完全可以满足需求，而且使用较为广泛，月夜不推荐安装其他的虚拟光驱。

解压缩软件

当然，除了虚拟光驱外，在安装大型软件之前，一个解压缩软件也较为必须，你可以选择免费且较为常用的7-Zip，不过月夜使用的是WinRAR，主要是使用的习惯性问题。而且月夜使用的这款软件还配备了自己比较喜欢的一款皮肤，可以给自己带来较好的视觉享受。下面是安装了这款皮肤之后的WinRAR软件的截图：

而下图是安装了这款主题后，Windows系统中压缩文件的图标：

这款皮肤名叫小小奕奕，如果你也喜欢的话，可以自行搜索下载安装。对了，最后特别提醒一点，虽然软件名为WinRAR，但从普遍意义上看，.zip的压缩格式却更为常用和通用，因此，推荐所有朋友为压缩文件使用.zip格式。

大型应用软件

做好了前期的准备工作，就可以为系统安装自己日常较为常用的大型应用软件了。如果你是程序员，则Visual Studio等IDE必不可少；如果你是家庭日用，则Photoshop等软件也较为必须；而如果你像月夜一样，需要从事科研工作，则MATLAB就不得不安装了。此外，Microsoft Office系列软件估计对所有人都不可或缺。不过这类软件的安装并没有什么需要特别说明的地方，月夜在此也不再赘言。

鼠标手势

自从在Chrome浏览器里爱上鼠标手势，尤其经StrokeIt扩大鼠标手势的适用范围后，月夜就开始对鼠标手势爱不释手。在新安装的系统里，自然不能少了鼠标手势的身影。 我们知道，Chrome浏览器里的鼠标手势只能在普通页面上使用，在新建的标签页上，在安全连接的页面里，都无法使用。可StrokeIt这款软件却将鼠标手势的使用范围进行了极幅地扩大，使得鼠标手势能在Chrome浏览器的所有页面上使用，此外，还能在应用软件里，在操作系统里进行使用，为我们提供了极大的方便。下面是月夜使用这款软件的界面图：

增强系统的文件搜索

虽然相较Windows XP，Windows 7操作系统对系统的搜索功能进行了增强，但我们在Windows 7中搜索硬盘上的文件时，仍然比较耗费时间，却不能满足我们对搜索的高效性需求。而一款软件Everything则很好地弥补了这一缺憾，该软件通过独特的文件索引机制，使得我们可以在硬盘上实时搜索到自己所要搜索的内容，而几乎不需要任何等待。下面是这款软件的界面图：

当然，为了在使用这一软件时获得最好效果，我们最好能把这款软件常驻系统，虽然需要耗费大约40M内存，但可以为我们提供实时搜索功能，也是十分值得。而另一方面，40M的内存消耗对而今动辄数G的内存配置来说，也可以忽略不计。

字体渲染的改进及系统美化

如你所知，微软在Windows 7操作系统中对系统美化上做了较大改进，使得我们的系统界面和字体越来越赏心悦目。然而，遗憾的是，由于微软采用的是基于TrueType的字体渲染方式，Windows 7操作系统下的字体显示效果依然不是那么美观。 而月夜在以前的文章中曾经介绍过一款对系统进行美化的软件，也就是众所周知的MacType，如那篇文章所言，使用这一软件对系统进行美化后，可以获得十分美观的效果。如下图所示是这款软件选择配置的界面：

不过需要说明的是，月夜在Windows 7系统中并没有对系统默认的字体进行替换，而是选择使用了雅黑和宋体，因为替换过之后在自己的电脑上看到的效果和最终获得的效果会有较大差距，另一方面，MacType对宋体渲染的效果也十分不错，方方正正看起来也讨人喜欢。

替换记事本

是不是受够了Windows系统中的记事本不能显示行号，不能实现自己想要的简单的功能？而使用写字板或者Word打开文档又有点兴师动众？那么来使用Notepad2吧，这款软件像记事本一样小巧，却可以实现众多记事本不能做到的功能，比如高亮当前行，比如语法高亮，比如代码折叠，等等。而更为关键的是，我们完全可以用这款小巧玲珑的软件替换掉系统自带的记事本，而使得我们应用起来得心应手。如下图是这款软件的界面：

可以看到相比系统自带的记事本，这款软件多么有亲和力。因此，下面我们就来看看如何使用这款软件来替换掉记事本。 可是下载完成这款软件后，在Windows 7操作系统中替换掉自带的记事本，因为Windows 7的用户权限管理机制，稍微有些麻烦。但核心只有这样一点，即使用Notepad2替换系统中如下位置的Notepad： C:/Windows C:/Windows/System32 C:/Windows/SysWOW64(此目录只存在64位的Windows) 而网上众多朋友提供了不少方法可供参考，但如果你图省事儿，而手头又恰巧有Win PE的启动U盘时，不妨通过U盘启动Win PE，而在Win PE中替换掉这三个位置的Notepad程序。

音视频播放器

以前的电脑上月夜使用的是千千静听，可是自从这款软件被百度收购以后，便朝向流氓化发展，而最为关键的是，它已经脱离了最初那款音频播放器的功能，加入了很多用户不需要的其他东西。此外，月夜在听音乐的过程中发现，千千静听播放起来的效果竟然不如QQMusic的效果好，因此只好将其放弃。 可是QQMusic也有些问题，虽然它很适合在线播放音乐，并且可以将自己正在播放的歌曲显示给QQ好友，但QQMusic中没有那种类似Shuffle的随机功能（比如可以把播放列表“随机乱序”，然后再选择“顺序播放”）。而QQMusic自己提供的随机播放功能，经常会在一个很短的时间内随机播放到同一首歌曲两次，会让你感觉很不爽，因此我也只好打消使用QQMusic来播放本地歌曲的打算。 剩下的只好使用Foobar2000了，可这款软件需要自己稍作打磨，经过整理后，月夜的Foobar2000的播放界面如下图所示：

当然，除了界面的美观，Foobar2000的播放效果已经为人熟知，月夜在此也不再赘言。略显遗憾的是，这款皮肤下，Foobar2000并没有直接把歌曲的码率等信息显示出来。 至于视频播放器，鉴于QQ影音越来越好的表现，其当然就成了月夜播放视频的不二选择。

截图软件

虽然我们在日常生活中更多使用的是QQ自带的截屏工具，但我们并不能保证电脑使用要挂着QQ，虽然QQ拼音输入法等也提供了截图工具，但我们显然更需要一款专业且小巧的截屏工具。而SPX恰恰是这样一款可以满足你随时随地截图需求的工具，这款软件的设置界面如下所示：

SPX提供了对窗口、矩形以及手绘区域等灵活的截图方式，此外还提供了注释、水印以及绚丽的边缘效果，对截得的图像，既可以直接保存到文件，也可以存入剪贴板里，还可以直接发送到编辑器里进行后期编辑，等等。总之，这款小小的软件，可以满足你几乎所有的日常截图需求；而软件只占用了十来M的内存，你甚至可以将其常驻到系统中随时使用。

浏览器

Windows 7中自带的浏览器已经升级为了IE 8.0，当然，你也可以将其升级为IE 10.0，这样，这款IE已经基本上能满足你的需求了，而且版本号到10.0的IE已经不再像6.0那样为人诟病，对标准的兼容已经相当不错。 无奈，在被IE“强奸”的岁月里，月夜已经深深地爱上了Chrome，因此，这款新系统中自然少不了Chrome的身影。 可是安装完Chrome浏览器之后，却遗憾的发现，在最新版的Chrome打开的新标签页中，8个“最常访问的网站”已经无法固定，而给我们带来了不少烦恼。此外，Chrome在不同电脑之间对浏览器数据和参数进行的同步实在是弱到曝。安装的扩展没有同步完全，有些扩展应用在新系统中消失不见；同步过来的扩展的设置选项参数也没有完全同步过来，还需要自己一步步的一遍遍进行设置。 当然，尽管存在这样一些问题，但Chrome的简单易用性还有速度还是让人欲罢不能，自然依旧是我浏览器的最爱。

图片查看工具

自从在以前的XP系统中开始使用Google的Picasa以后，月夜就对Windows照片查看器嗤之以鼻，无论是易用性，还是绚丽的效果上，Picasa都带给了我们不一样的感受。而且，Picasa还提供了图片同步到云端功能，可以防止由于自己的电脑损坏而造成照片丢失。此外，Picasa提供的图片汇总与归类，人像识别等功能，也实在让人大爱。 不过遗憾的是，就像Google的Chrome浏览器一样，Picasa不支持对gif等动态图片的查看，使用Picasa查看这类图片时，会跟Chrome浏览器一样，看到的只是一副静态效果图。
