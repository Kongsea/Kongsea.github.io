美化Windows系统显示效果

Posted on 2011-03-13 | In 精品推介

众所周知，Windows系统下的界面显示效果不甚美观，更不用说与Mac系统相提并论了。使用Mac系统时，其界面或许是一种享受，但使用Windows系统的我们却不得不忍受其不堪的字体与丑陋的界面。 然而，当MacType横空出世时，我们终于可以摈弃Windows难受的字体和界面转而将其美化并享受这可以和Mac系统下相媲美的界面了。 让我们先来看一看简单安装并启用MacType，甚至不需要进行专门的设置，经此美化之后，使用浏览器访问月夜博客的首界面时，已经是怎样的一种美观了：

再来看看我们最常用的聊天工具，QQ的界面：

怎么样？是否已经被这样美观的界面和字体征服了？那么赶快跟随月夜一起来把你的系统也美化一下吧。

1.下载MacType

随便在Google上搜索一下，就能找到无数的MacType下载链接。而该软件并不大，只有区区3M而已，因此下载起来也不会太困难。 这是MacType作者的官方主页，如果诸位不放心，可以直接访问官方主页下载软件，或者关注MacType的最新消息：MacType官方主页。

2.安装MacType

安装MacType这样的小软件更不是什么难事，基本上可以一路next点下去；当然，如果你关心细节，想要自己定制安装位置等信息，则需要在安装中稍作修改。 安装中需要注意的地方是在最后的向导中，在“请选择适合您的MacType加载方式”中，因为我们尚为新手，所以最好选择“MacType托盘加载”和其下的“独立加载模式”。而在向导的下一步“请选择适合您的配置文件”中，则可以根据自己的需要已经自己的显示器来选择一款最适合自己的配置了。月夜选择的是“XMac.LCD.HotShift@XHei(iOS^MacOSX)”，不过放心，以后你可以随时方便地更换或重新选择配置文件。

3.使用MacType

安装设置到这里，点击完成，就可以看到最后优化的完美效果了。可能需要重启，也可能需要重新选择不同的配置文件，很轻松地，你就可以得到最适合自己，也是自己最喜欢的美化效果。 当然，如果你比较喜欢DIY，还可以通过MacType进行更为灵活地设置，打造属于自己的配置文件。

4.高级自定义

这里所说的是使用另一款软件辅助MacType来打造完美的美化效果，这款软件的名字叫SystemFontsReplacer。如名所言，该软件实现替换系统字体的功能。具体到这里，我们使用这款软件将系统中的宋体和新宋体替换掉。 需要注意的是，使用通常的字体来替换Windows系统中的宋体会发生乱码现象，主要是因为通常意义上的字体没有宋体所包含的字库全。为了不出现乱码，我们可以使用极限论坛打包的雅黑字体。这里是这款字体的下载地址。 下载解压后，就可以使用SystemFontsReplacer替换掉系统的宋体了，如下图所示：

替换掉以后，在我们的Windows系统中就可以跟讨厌而丑陋的宋体说拜拜了。 不过，因为宋体对Windows系统的重要性，对不是特别追求美观性的朋友，月夜不建议将系统中的宋体替换掉。好了，关于MacType的介绍，月夜就为你介绍到这里，因为我们相信，仅仅这些已经足够让你将系统做一次不错的美化；而关于MacType的更详细介绍，你可以参阅这篇博文。

最后，特别感谢QQ好友Max为月夜推荐MacType，并协助我们使用。

2013年3月更新：

经月夜使用发现，在Windows 7操作系统下使用微软正黑体(Microsoft JhengHei)会得到意想不到的更好效果，推荐朋友们使用。
